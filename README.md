## Задача:
### Сделать калькулятор оценок, который расчитывает средний балл, переводит из 100 балльной шкалы в 5 балльную и определяет сдан ли экзамен по оценке, а так же меню с выбором

## Функции программы:
- menu() - Основное меню навигации
- avg_grades(grades) - Средний балл оценок
- convert(grade) - Перевод из 100 балльной шкалы в 5 балльную
- pass_exam(grade, pass_score = 50) - Чекает сдан ли экзамен по оценке
- calculation(grades) - Основные данные расчета
- random_grade(count = 10) - Рандом оценки
- load_file(filename = "data.txt") - загрузка данных из файла data.txt
- save_file(results,filename = "result.txt") - выгрузка данных в файл result.txt
- print_res(results) - Вывод результатов на экран
